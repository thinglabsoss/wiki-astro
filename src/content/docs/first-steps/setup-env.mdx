This page provides step-by-step instructions to set up your system for flashing your Car Thing, from tool preparation to software configuration for a successful process.

## Installing Programs
The flashing process requires some items to be successful

### Git
<Tabs syncKey="system">

  <TabItem label="Windows" icon='seti:windows'>
    Install Git on your system if is not available [here](https://git-scm.com/downloads)
  </TabItem>

</Tabs>


### Python

Python is used to run the Car Thing flasher 

<Tabs syncKey="system">

  <TabItem label="Windows" icon='seti:windows'>
    Download and Install the lasted python version [here](https://www.python.org/downloads/)

    <Aside>
    Don't forget to check Add Python to PATH option on the installer as menitoned [here](https://docs.python.org/3/using/windows.html#installation-steps)
    </Aside>
  </TabItem>

  <TabItem label="Mac" icon='apple'>
    Install [Homebrew](https://brew.sh/) and python will come along.

     <Aside >
      Homebrew is necessary since the macOS default python installation dosen't work with the installer
    </Aside>
  </TabItem>

  <TabItem label="Linux" icon='linux'>
    <Aside type='tip'>What did you think linux is great!</Aside>
  </TabItem>

</Tabs>

### Drivers

To recognize the Car Thing is necessary some specific drivers

<Tabs syncKey="system">
  <TabItem label="Windows" icon='seti:windows'>
    Plug in the USB end of the Car Things USB C cable into the computer, then hold top buttons 1, and 4 and plug in the USB C into the Car Thing, this will put it in USB burning mode.

    For Windows to recognize the Car Thing you need Zadig, you can get it from [here](https://zadig.akeo.ie). After its installed click options then “list all devices”.

    ![zadix_options](/src/assets/docs/zadix_ignore.png)

    Then change it to “GX-CHIP” and click the little arrows to change it to LibusbK, then click install and wait for it to finish.

    ![zadix_libusbk](/src/assets/docs/zadix_libusbk.png)

    <Aside type='caution'>If this driver won't work try using WinUSB.</Aside>
  </TabItem>
  
  <TabItem label="Mac" icon='apple'>
    ```zsh
    brew install libusb
    ```
  </TabItem>

  <TabItem label="Linux" icon='linux'>
    With your package manager of choice install the packages
    ```
    libusb libusb-compat
    ```
  </TabItem>
</Tabs>
