A simple app that adds support for more windows media playback data. To install it, simply download the application from the **Downloads** tab in the DeskThing Server, see [Managing Apps](/official-apps/deskthing/managing-apps).

You can change your Car Thing playback sources under **Settings** in the **Music** tab.
